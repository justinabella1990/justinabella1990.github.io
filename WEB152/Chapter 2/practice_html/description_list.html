Sample Description

TCP
Transmission Control Protocol is a set of rules that govern the way data is transmitted over the internet, ensuring that data packets are delivered in order and with minimal loss.
IP
Internet Protocol is the primary protocol that directs the flow of data packets over the internet, responsible for addressing and routing data from one computer to another.
FTP
File Transfer Protocol is a standard protocol used to transfer files between computers over the internet, used mainly for uploading and downloading files.
HTTP
Hypertext Transfer Protocol is the foundation of the World Wide Web, used for transmitting data over the internet. HTTP defines how messages are formatted and transmitted, and what actions Web servers and browsers should take in response to various commands.
